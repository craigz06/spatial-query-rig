# Query Types

## The Comparison

Two conditions. Same question. Same environment. Same moment.

**Condition A — Stateless text query**  
Standard prompt to an AI model with no environmental context.
No spatial reference. No session history. No physical anchor.

**Condition B — Grounded stereo sequential query**  
Query delivered as stereo sequential frames from the Sentinel Rig.
Model has access to accumulated session files from the same
physical environment. Known room geometry. Known human. Known depth.

---

## What Changes

In Condition B the model is not answering from nowhere.
It is answering from a known place, with a known history,
observed by a known instrument.

---

## The Measurement

Outputs from both conditions compared on:

- Accuracy of spatial reference
- Contextual relevance
- Resolution of ambiguity
- Depth of response to the same question
